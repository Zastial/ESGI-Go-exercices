Questions à explorer :

```go
// Que retourne ce code ?

var s string

fmt.Printf("%q\n", s)

// Quelle est la différence ?

x := 0

var y int
```

Réponses :
1) Le code retourne une chaîne de caractères vide : "".

2) La variable x est initialisée à 0, tandis que la variable y est déclarée mais non initialisée, il n'y a pas de réelle différence entre les deux.
